1.  Start Eclipse IDE

2.  Under File->New->C Project.  Allow it to use the default workspace, and name the
project HexMerge.

3.  Highlight the project directory, right-click,TortoiseSVN->Export.
Under URL of repository:  Choose https://open-pinball-project.googlecode.com/svn/trunk/Eclipse/0202-HexMerge.
This extracts all the files from the repository.

4.  In eclipse, choose File->Refresh

5.  In eclipse, choose Project->Build All

6.  This creates the executable file in project directory\Debug\HexMerge.exe
